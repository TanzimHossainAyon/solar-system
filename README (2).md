# 🌌 Solar System Simulator

An interactive 3D-style solar system simulator built with Canvas API and vanilla JavaScript.

## 👤 Author
**Tanzim Hossain Ayon** | CSE Graduate · BRAC University · IEEE Researcher
🌐 [Portfolio](https://tanzimhossainayon.vercel.app)

## 🚀 Features
- ☀️ All 8 planets orbiting the Sun with real speed ratios
- 🪐 Saturn's rings rendered with ellipse geometry
- 💫 Shooting stars & twinkling star field
- 🌌 Nebula background with ambient glow
- ☄️ Animated asteroid belt between Mars & Jupiter
- 🖱️ Click any planet for detailed info panel
- ⚡ Speed control (0.1x to 5x)
- ⏸️ Pause/Resume animation
- 🔍 Scroll to zoom in/out
- 🖐️ Drag to pan the view
- 📱 Touch support for mobile
- 💾 Speed setting saved via LocalStorage

## 🛠️ Technologies Used
| Technology | Purpose |
|---|---|
| HTML5 Canvas API | Rendering planets & animations |
| RequestAnimationFrame | Smooth 60fps animation loop |
| Math Trigonometry | Orbital calculations |
| CSS Variables | Dynamic theming |
| LocalStorage API | Saving speed preferences |
| Web Events API | Mouse, touch, scroll interactions |
| CSS Animations | UI panel & float effects |
| Radial Gradients | Realistic planet shading |

## ▶️ How to Run
Open `index.html` in any browser — no installation needed!

## 🎮 Controls
- **Click** planet → View planet info
- **Scroll** → Zoom in/out
- **Drag** → Pan the view
- **Speed slider** → Control orbital speed
- **Pause** → Freeze animation
